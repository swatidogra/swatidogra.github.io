Projects

  • Amazon Stock Price Prediction Using Machine Learning Techniques - This project aims to predict Amazon's future stock prices using historical data and machine learning techniques to assist investors and financial analysts in making informed decisions. Amazon's stock price history, sourced from NASDAQ, includes daily data over the past five years. The project employs Linear Regression, ARIMA, and LSTM models, each with different strengths and weaknesses in forecasting accuracy. The study emphasizes the need for clean, accurate data and considers challenges such as model overfitting and market volatility while providing recommendations and an implementation plan for effective model usage.
