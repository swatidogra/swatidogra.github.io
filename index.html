Projects

  • Amazon Stock Price Prediction Using Machine Learning Techniques - This project aims to predict Amazon's future stock prices using historical data and machine learning techniques to assist investors and financial analysts in making informed decisions. Amazon's stock price history, sourced from NASDAQ, includes daily data over the past five years. The project employs Linear Regression, ARIMA, and LSTM models, each with different strengths and weaknesses in forecasting accuracy. The study emphasizes the need for clean, accurate data and considers challenges such as model overfitting and market volatility while providing recommendations and an implementation plan for effective model usage.
  • Understanding and Addressing Childcare Costs and Socioeconomic Impacts in the U.S. - This analysis examines childcare costs across different regions in the United States, highlighting their impact on household budgets and related socioeconomic factors. Key findings include significant cost variations across states and age groups, persistent gender income disparities, and differences in unemployment rates and labor force participation. The study emphasizes the need for affordable childcare options and aims to raise awareness, advocate for supportive policies, and provide practical insights for managing childcare costs. The target audience includes parents, policymakers, childcare providers, and advocacy groups, with information presented through engaging mediums such as videos, infographics, and presentations.
  • Predictive Modeling of Cosmetics Product Ratings Using Machine Learning - This project aims to predict cosmetics product ratings using machine learning models such as Logistic Regression, K-Nearest Neighbors (KNN), and Random Forest Classifiers. Leveraging a dataset from Kaggle, the analysis identified key features like price, brand, and skin type suitability that impact ratings. The models were evaluated using metrics like ROC AUC scores, with Logistic Regression. Findings indicate that accurate prediction of product ratings can enhance consumer satisfaction, aid brands in improving products, and maintain market competitiveness. Recommendations include further feature engineering, model tuning, and exploring advanced algorithms to refine predictions and personalize recommendations.
  • Predictive Sales Forecasting for E-Commerce Platforms Using Advanced Data Science Techniques - Predicting future sales is crucial for inventory management, marketing strategies, and financial planning on e-commerce platforms. Accurate sales forecasting helps optimize stock levels, reduce costs, plan marketing campaigns, and enhance customer satisfaction by considering peak seasons and hours. By leveraging advanced data science techniques, such as exploratory data analysis (EDA), data preprocessing, and machine learning models like Linear Regression, Logistic Regression, and K-Nearest Neighbors (KNN), businesses can gain valuable insights into sales patterns. This project aims to predict sales using historical data, sourced from Kaggle, which includes features such as product category, price, order date, and sales, ultimately improving resource allocation, pricing strategies, and overall profitability.
  • Advanced Travel Recommendation System Using Data Science Techniques - This project aims to create a travel recommendation system using data from flat files, APIs, and websites, recommending hotels, tours, and attractions based on user preferences and location. The project involves data preprocessing, integration, and visualization. Key challenges include ensuring data quality and consistency, handling varying data volumes, and building a scalable database. Ethical considerations involve addressing potential biases in hotel reviews and ensuring the fairness of the recommendations. Data science techniques used include data cleansing, standardization, and visualization to derive actionable insights for travel recommendations.
  • Analyzing and Predicting Trends in the Electric Vehicle Industry Using Data Science Techniques - This project analyzes the Electric Vehicle Population dataset from Kaggle to uncover insights that can inform decision-making in the electric vehicle sector. Through rigorous data cleaning, exploratory data analysis, statistical testing, and predictive modeling, the study reveals trends in EV growth, popular makes and models, and the distribution of different types of EVs. By forecasting future trends, the project aims to support informed decision-making for consumers and stakeholders. While limitations exist, the analysis provides a strong foundation for further research and real-world applications in promoting electric vehicle adoption.
  • Comprehensive Analysis of Pregnancy and Birth Outcomes Using NSFG Data - This project involves analyzing the NSFG dataset to uncover insights related to pregnancy and birth. Key variables analyzed include age during pregnancy, birth order, pregnancy length, total birth weight, and birth weight for live births. The project includes creating histograms to visualize data distribution, examining potential correlations between variables using scatter plots, and performing regression analysis. Through exploratory data analysis and statistical methods, the project aims to understand central tendencies, variability, and relationships within the dataset, providing a comprehensive understanding of the factors influencing pregnancy and birth outcomes.
  • Data Extraction and Visualization Project - This project involves various data extraction and visualization tasks. One task extracts the top 100 eBooks from the Gutenberg website using HTML parsing techniques. Another task is to build a movie database by fetching data from the OMDB API. Additionally, a task connects to an API to pull and extract specific data, such as artwork titles and artist names. Finally, the project includes plotting various charts using a given dataset to visualize data insights comprehensively.
